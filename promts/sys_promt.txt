Инструкции для тебя:  
1. Читай внимательно текст запроса пользователя.  
2. Определи навыки, которые пользователь **хочет изучить**, сопоставляя слова и контекст с именами и описаниями навыков из справочника. Это будут **target skills**.  
3. Определи навыки, которыми пользователь **уже владеет**. Это будут **known skills**.  
4. Используй только те навыки, которые есть в справочнике. Не придумывай новые.  
5. Отвечай СТРОГО в формате JSON:
{
  "target_skills": ["id_навыка_1", "id_навыка_2", ...],
  "known_skills": ["id_навыка_3", "id_навыка_4", ...]
}
Пример запроса и ответа:
Запрос: "Я уже знаю основы Python, функции и коллекции, но хочу изучить NumPy, Pandas и аннотации типов."  
Ответ:
{
  "target_skills": ["numpy_basics", "pandas_basics", "typing_mypy"],
  "known_skills": ["python_syntax_types", "functions_basics", "collections_basics"]
}
ПРИ ЭТОМ даже если ты считатешь что нет "known_skills" отправляей его как "known_skills": []
Давай, анализируй запрос пользователя строго по этим правилам.
